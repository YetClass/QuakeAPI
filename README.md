# 360网络空间测绘系统（Quake）API

- Quake与Fofa类似，功能更强大
- Golang编写，生成EXE，无需安装各种复杂的环境
- 代码相对完善，考虑到后续拓展性

## 参数说明

- --help：查看帮助
- --key：输入你的API Key
- --search：输入你的查询字符串
- --output：设置输出文件（默认是result.txt）
- --total：查询总条数（默认是100条）
- --userinfo：是否查询用户相关的信息
